# Simply

Command line tooling for SimplicityHL.

## Installation

```sh
cargo install --git https://github.com/m-kus/simply simply
```

## Usage

### Build

```sh
simply build main.simf
```

### Run

```sh
simply run main.simf
```

### Debug

```sh
simply debug main.simf
```
