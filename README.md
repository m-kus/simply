# Simply

Command line tooling for SimplicityHL.

## Installation

```sh
cargo install --git https://github.com/m-kus/simply simply
```

## Usage

### Build

```sh
simply build --entrypoint main.simf
```

### Run

```sh
simply run --entrypoint main.simf
```

### Test

```sh
simply test
```

### Deposit

```sh
simply deposit
```

### Withdraw

```sh
simply withdraw
```
